By the end of the project, I already had a complete library of ternary and binary logic circuits including others' work and my own.
This suite of schematics that include THA, TNAND, TOR, STI etc. is available through the git repository, though compatibility isn't guaranteed.

The STI designs and TNAND schematic designs require only the two CNFET models and are the most likely to be portable.
